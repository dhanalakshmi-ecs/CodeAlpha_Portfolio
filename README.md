# Personal Portfolio Website

This repository contains my personal portfolio website developed using HTML and CSS.
The purpose of this project is to present my profile, technical skills, and projects
in a clean and professional manner.

This portfolio was created as **Task 3** of the **CodeAlpha Frontend Development Internship**.

---

## Project Overview

A portfolio website is an essential tool for showcasing skills and projects.
This website provides a structured overview of who I am, what technologies I work with,
and the projects I have built during my learning journey.

The design focuses on simplicity, readability, and responsiveness to ensure a good
user experience across different devices.

---

## Features

- Professional hero section with name and role
- About Me section describing background and interests
- Skills section displayed in a clean grid layout
- Projects section highlighting completed web projects
- Contact section with email and professional links
- Responsive layout suitable for desktop and mobile screens
- Clean UI with consistent color theme

---

## Technologies Used

- **HTML5** – Used to structure the content of the website
- **CSS3** – Used for styling, layout, responsiveness, and visual design

No external frameworks or libraries were used, keeping the project simple and lightweight.

## Sections Explained

### Hero Section
The hero section introduces my name and professional role.
It creates a strong first impression and includes a call-to-action button
to navigate directly to the projects section.

### About Me
This section provides a brief introduction about my background, interests,
and learning goals as a frontend developer.

### Skills
The skills section highlights the technologies and tools I am familiar with.
It is displayed in a grid format for better readability.

### Projects
This section showcases the projects I have completed, including:
- Image Gallery project
- Calculator Web Application

Each project is briefly described to explain its purpose and features.

### Contact
The contact section includes my email address and links to my GitHub and LinkedIn
profiles, allowing easy professional communication.

## How to View the Project

1. Download or clone this repository
2. Open the `index.html` file in any modern web browser
3. Navigate through the sections using scrolling

No additional setup or installation is required.
